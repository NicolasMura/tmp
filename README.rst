===============================
La bibliothèque de who you know
===============================

.. image:: HelloHenri.png

Online demo : http://nicolasmura.fr/expelliarmus/dist/hello-henri

Stack utilisée
==============

* NodeJS : 10.16.0
* Angular: 8.0.2
* Angular Cli : 8.0.4

Installation
============

.. code-block:: shell

  npm i
  ng serve

MEP (accès SSH requis)
======================

.. code-block:: shell

  npm run deploy-prod
